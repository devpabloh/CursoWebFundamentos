Declarações condicionais

Declarações Se (if)

Instrução que verifica uma condição para decidir se será executado ou não, aquele bloco de códigos

As declarações if...else nos permitem automatizar soluções para perguntas do tipo sim ou não, também conhecidas como decisões binárias.

A declaração If é composta por

  • A if palavra-chave seguida por um conjunto de parênteses () que é seguido por um bloco de código, ou instrução de bloco, indicado por um conjunto de chaves{}
  • Dentro dos parênteses (). É fornecida uma condição avaliada como True ou False.
  • Se a condição for avaliada como true. O código entre chaves {} será executado.
  • Se a condição for avaliada como false, o bloco não será executado.

Declaração If(se) e Else(caso contrário)

Se quisermos adicionar algum comportamento padrão à instrução If, podemos adicionar uma instrução Else para executar um bloco de código quando a condição for avaliada como false.

  • Usa-se a palavra-chave else após o bloco de código de uma instrução if(se).
  • Possui um bloco de código envolvido por um conjunto de chaves {}.
  • O que está dentro do bloco de código {} da instrução else será executado quando a condição da instrução if for avaliada como false.

Declarações Else if

Podemos adicionar mais condições ao nosso IF e else com uma declaração else if. A declaração else if permite mais de dois resultados possíveis, podendo ser adicionadas quantas instruções else if você desejar.

A declaração else if sempre vem depois da declaração if e antes da declaração else. A declaração else if também requer uma condição. Vamos dar uma olhada na sintaxe.

Operadores de comparação

Ao escrever instruções condicionais, às vezes precisamos usar diferentes tipos de operadores para comparar valores. Esses operadores são chamados de operadores de comparação.

A seguir uma lista de alguns operadores de comparação úteis a sua sintaxe:

  • Menor que: <
  • Maior que: >
  • Menor que ou igual: <=
  • Maior que ou igual: >=
  • É igual a: ===
  • diferente de: !==

Operadores lógicos

Trabalhar com condicionais significa que usaremos booleanos que tem como valores true ou false. Em javaScript existem operadores que trabalham com valores booleanos conhecidos como operadores lógicos.

Existem três operadores lógicos:

  • O operador lógico E que é representado por &&
  • O operador Ou que é representado por || (para colocar esse sinal, ele fica geralmente próximo ao shift do teclado)
  • O operador not, ou operador de negação que é representado por !

Usamos o operador && quando queremos saber se as duas condições são true

Usamos o operador || quando precisamos que apenas uma das condições seja true

usamos o operador ! para negar ou inverter a negação de um valor em booleano
