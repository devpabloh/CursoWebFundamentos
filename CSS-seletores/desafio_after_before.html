Desafio

  1. Incluir ícone DEPOIS do texto dos links que abrem em nova aba;
  2. Incluir ícone Antes do texto dos links que abrem um PDF; e
  3. Colorir os textos dos labels que vêem após os checkboxes seguindo a seguinte regra:
    1. Após o status checked a cor do label deve ser verde;
    2. Após o status não cheched a cor do label deve ser vermelho; e
    3. Se o label vem após um background qualquer no terceiro filho(li) desde que esteja contido dentro de outra lista(ul ou ol).

Atenção: Não é permitido alterar nada no HTML (nem mesmo adicionar classes).
