Função

Em javaScript, uma função é um bloco de código designado para realizar uma tarefa especifíca. Elas são usadas para agrupar um conjunto de instruções para que possam ser reutilizadas facilmente em diferentes partes do código. As funções podem ser chamadas (ou "invocadas") pelo nome, e podem aceitar parâmetros(dados que podem ser passados para função) e retornar um valor
